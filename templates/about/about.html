About the Project

Advanced height prediction model using deep learning, integrating genetic, environmental, and lifestyle factors with robust preprocessing for accuracy, including data normalization, feature selection, and outlier handling for improved model performance and generalization.

Project Goals

  • Enhance prediction accuracy using advanced feature engineering.
  • Integrate genetic, lifestyle, and socioeconomic factors.
  • Improve healthcare outcomes with reliable growth tools.
  • Deliver robust models through data preprocessing methods.

Technologies Used

  • Machine Learning Models : Linear Regression, SVR, XGBoost, LightGBM
  • Framework : Flask
  • Programming Languages : Python
  • Datasets : Galton's height data
